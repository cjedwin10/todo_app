# Daily Todo App with Notifications 📝

A simple, mobile-friendly todo list application built with Flask and SQLite. Get reminders for your daily tasks with browser notifications!

## Features ✨

- User authentication (register/login)
- Create daily tasks with specific times
- Get notifications 5 minutes before a task starts
- Receive "Did you complete?" prompts for past tasks
- Yes/No response handling
- Mobile-responsive design
- Works offline on your phone!

## Tech Stack 🛠️

- **Backend**: Python, Flask
- **Database**: SQLite
- **Frontend**: HTML5, CSS3, JavaScript
- **Notifications**: Browser Notification API
